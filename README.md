# 🎮 Ignora Store

A cyberpunk-themed digital game store for Android.  
Built with Flutter using Firebase Auth, Cloud Firestore, and Supabase Storage.

---

## ✨ Features

- 🎮 Browse and discover games
- 🔐 Secure authentication
- 💳 Purchase and download games
- ⭐ Reviews and ratings
- 🔔 Real-time notifications
- 👨‍💼 Admin dashboard
- 🌍 Multi-language (EN / FR / AR)
- 🎨 Cyberpunk neon UI

---

## 📦 Installation

1. Download the APK
2. Enable **Install from unknown sources**
3. Install and enjoy 🚀

---

## 🛠️ Tech Stack

- Flutter
- Firebase Authentication
- Cloud Firestore
- Supabase Storage
